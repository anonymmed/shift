# SHIFT

This project is created by Symfony framework for backend and Angular for frontend.

## Installation
#### Note: You need NODE, NPM and Composer to use this project
In the backend folder run.

```bash
composer install
```


In the frontend folder run.

```bash
npm install
```
## Usage
For Future features, a software engineer will be able to get the user's submission using the api:
```
https://52.23.240.171:8000/api/v1/GetUserAnswers/{email}
```
And for user's MBTI result :
```bas
https://52.23.240.171/api/v1/GetUserMBTIScore/{email}
```
